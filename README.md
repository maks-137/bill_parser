Опис
------
Парсер даних законопроектів з [сайту Верховної Ради України](https://itd.rada.gov.ua/billInfo)


Встановлення
-------
```
git clone https://github.com/maks-137/bill_parser
cd bill_parser
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```


Запуск
-----
```
python main.py        
```
